# burger-maker

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start
```
#### Note:
- Make sure to install Node.js v11.14.0 to make sure the app runs fine. Testing for other node versions is yet to be done.
